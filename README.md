# fer (Facial Expression Recognition )

## Overview 
画像から表情を予測するプログラム（正答率は約30%)

## Requirement
- Python3

##Usage
Kaggle(FER-2013)のファイルをダウンロードして、csvファイルから画像ファイルに変換後、モデル構築、学習、予想させる。
